Quartz

2025-02-11

📚 공식 문서 💻 Github Repository 💬 Discord Community

Quartz는 개인의 디지털 노트나 디지털 가든을 웹사이트로 손쉽게 배포할 수 있게 도와주는 오픈소스 정적 사이트 생성기임.

특징

---

배터리 포함

  • Quartz를 관통하는 철학
  • 필요로 할 수 있는 기능들은 추가 설치 없이 바로 사용 가능하게 설계
  • 완성도 있는 활용 가능한 패키지들이 모두 포함되어있다는 뜻
  • 예시
  • Markdown <-> HTML 변환 플러그인
  • Obsidian 노트의 링크 처리
  • 백링크 생성
  • 그래프 뷰 등...

Obsidian Publish 무료 대안

  • Quartz 4.0은 Obsidian의 유료 퍼블리싱 서비스와 달리 무료인 오픈소스 프로젝트임

Node.js 기반

  • 4.0 이전엔 Hugo 라는 정적 사이트 생성기 기반이었으나, 4.0 이후엔 Node.js를 엔진으로 하여 사이트 빌드와 퍼블리싱 작업을 수행함
  • Node.js를 사용함으로써 JSX 지원 및 상세한 오류 메시지와 원활한 사용자 경험을 제공

시작하기

--- 해당 시작하기는 공식 홈페이지

환경 구성

  • Node.js v20 이상
  • npm v9.3.1 이상
  • git

프로젝트 생성

git clone https://github.com/jackyzha0/quartz.git
cd quartz
npm i
npx quartz create

기타 가이드

	npx quartz build # 빌드, 별도 output 지정이 없는 경우 ./public에 빌드됨
	npx quartz build -d /another/path # content 경로 변경하여 빌드
	npx quartz build -v # 빌드 시 다양한 로그 정보가 남음
	npx quartz build -o /another/path # 빌드 결과물이 저장될 폴더, 기본값은 ./public
	npx quartz build --serve # local 환경에서 프로젝트 미리보기
	npx quartz build --serve --port 9999 # 프로젝트 미리보기 포트 지정(기본 8080)
	npx quartz build --concurrency 3 # 노트를 분석하는데 몇개의 쓰레드를 사용할건지 (기본 1)

기능

  • Obsidian 호환성
  • 전체 텍스트 검색
  • 그래프 뷰

  • LaTeX (수학 공식) 지원
  • 구문 강조 (Syntax Highlighting)
  • 팝오버 미리보기
  • 링크 위에 마우스를 올리면 해당 페이지의 제목, 메타데이터, 내용 일부 등이 보임
  • Docker 지원
  • 다국어 지원 (Internationalization)
  • 댓글 기능 및 그 외 많은 기능들이 기본 제공됨
  • giscus를 통한 댓글 지원
